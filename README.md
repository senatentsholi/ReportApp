# Limkokwing Faculty Reporting App

Mobile reporting application for Limkokwing University of Creative Technology.

## Stack

- React Native with Expo
- Firebase Authentication and Firestore
- React Navigation
- React Native Paper

## Features

- Role-based login and registration
- Student monitoring, ratings, and attendance views
- Lecturer class management, attendance capture, and weekly reporting
- Principal Lecturer course, report, class, monitoring, and rating views
- Program Leader course and class management, reporting, monitoring, and rating views
- Search across the main modules
- Excel export for report records

## Getting Started

1. Install dependencies:

```bash
cmd /c npm.cmd install
```

2. Start the project:

```bash
cmd /c npm.cmd start
```
